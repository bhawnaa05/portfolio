SkillSphere — The Barter-Based Skill Exchange Platform

SkillSphere is a community-driven skill-sharing platform that enables students to teach and learn directly from one another using a barter-style system—without paying for expensive courses.

Key Features:

  • Skill matching system that pairs users based on what they can teach vs. what they want to learn
  • Real-time chat to schedule and coordinate sessions
  • Google Meet API integration for auto-generated learning sessions
  • Predefined skill categories for easy filtering and searching
  • Rating & feedback system to build trust in the community
  • Fully responsive design with modern UI

Tech Stack: React, Bootstrap, Node.js, Express, MongoDB, Google Meet API

Designed to solve the lack of affordable, interactive education by promoting a peer-to-peer learning culture, SkillSphere empowers learners through community collaboration and mutual growth.
