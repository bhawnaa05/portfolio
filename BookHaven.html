BookAlogy is an interactive platform for book lovers to explore, discover, and purchase their favorite reads with ease. It offers a smooth, user-friendly experience with engaging design and intuitive navigation, making book browsing more enjoyable and accessible.

Key Features Implemented:

  • Responsive and mobile-friendly UI with intuitive navigation
  • Advanced book search and filtering by genre, author, and rating
  • User authentication system with secure login and registration
  • Shopping cart and dynamic checkout flow
  • Admin dashboard to manage books, orders, and users
  • Wishlist and save-for-later functionality for logged-in users
  • Ratings and reviews section for community feedback

Technical Stack:

  • Front-End: React.js, Bootstrap
  • Back-End: Node.js, Express.js
  • Database: MongoDB
  • Deployment: Vercel (Frontend), Render (Backend)

BookAlogy aims to make book discovery joyful and convenient, offering both casual readers and book collectors a clean, fast, and organized shopping experience.
